# 🐍 Игра «Змейка» с сохранением рекордов

Классическая игра, дополненная серверной частью для хранения топ-10 рекордов в базе данных. Проект выполнен в рамках производственных практик **ПМ.02** (интеграция модулей) и **ПМ.11** (разработка, администрирование и защита БД).

## 🚀 Демо

https://ivanba1.github.io/hexlet-praktika-zmeika/ – замените после выкладки.

## 📦 Стек технологий

- **Frontend:** HTML5, CSS3, JavaScript (Canvas)
- **Backend:** Node.js, Express
- **База данных:** SQLite (может быть заменена на PostgreSQL)
- **Деплой:** Vercel (frontend) + Render / Heroku (backend)

## 📈 Система подсчёта очков
Каждое съеденное яблоко приносит +10 очков.
Счёт отображается в реальном времени над игровым полем.
